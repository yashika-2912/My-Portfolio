<h1><mark>Simple HTML Portfolio Card</mark></h1>

<p>
<b>Simple HTML Portfolio Card</b> is a beginner-friendly personal portfolio project
built using <b>HTML</b> and <b>CSS</b>. It is designed to help beginners understand
basic web development concepts through a clean and minimal
<mark>card-based layout</mark>.
</p>

<hr>

<h2><b>Project Description</b></h2>
<p>
This project presents personal and professional details in a single portfolio card.
It is ideal for <mark>students and beginners</mark> who are starting their web
development journey.
</p>

<hr>

<h2><b>Portfolio Includes</b></h2>
<ul>
  <li><b>Student Name</b> and <b>Role</b></li>
  <li><b>About Me</b> section</li>
  <li><b>Technical Skills</b></li>
  <li><b>Projects Completed</b></li>
  <li><b>Areas of Interest</b></li>
  <li><b>Contact Details</b></li>
</ul>

<hr>

<h2><b>Features</b></h2>
<ul>
  <li><mark>Simple and clean UI</mark></li>
  <li><b>Card-based layout</b></li>
  <li><b>Easy to customize</b></li>
  <li><b>Beginner-friendly code</b></li>
  <li><mark>Responsive-ready structure</mark></li>
</ul>

<hr>

<h2><b>Technologies Used</b></h2>

</ul>
