**Simple HTML Portfolio Card**

It is a very simple personal portfolio card built using HTML and CSS. It is designed as a beginner-friendly project to showcase basic personal details, skills, projects, and contact information in a clean card layout.

**The portfolio includes:**

Student name and role

Short About Me section

List of technical skills

Projects completed

Areas of interest

Contact details

**Features**

Simple and clean UI

Card-based layout

Easy to customize
